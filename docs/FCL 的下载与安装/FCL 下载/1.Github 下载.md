> FCL在GitHub开源，GitHub是下载FCL**最正规、最安全的**渠道，你可以在FCL的[GitHub Release](https://github.com/FCL-Team/FoldCraftLauncher/releases)页面下载它的最新版和所有历史版本。

## 步骤：

1. 浏览器访问FCL的GitHub Release页面![1.访问GitHubRelease](../../../static/img/docs/FCL%20的下载与安装/FCL%20下载/GitHub%20下载/1.访问GitHubRelease.jpg)
2. 下划找到绿色的**Latest**字样![2.找到Latest](../../../static/img/docs/FCL%20的下载与安装/FCL%20下载/GitHub%20下载/2.找到Latest.jpg)
3. 找到Latest对应的**Release**![3.找到Latest对应的Release](../../../static/img/docs/FCL%20的下载与安装/FCL%20下载/GitHub%20下载/3.找到Latest对应的Release.jpg)
4. 找到该Release的**Assets**![4.找到Assets](../../../static/img/docs/FCL%20的下载与安装/FCL%20下载/GitHub%20下载/4.找到Assets.jpg)
5. 选择并点击**适合你的**FCL版本![5.找到合适的版本](../../../static/img/docs/FCL%20的下载与安装/FCL%20下载/GitHub%20下载/5.找到合适的版本.jpg)
6. 等到下载完成并安装

:::tip 无法访问/下载？
GitHub因为国内防火墙（GFW）的原因在绝大多数情况下无法访问，你可能需要使用**代理**来访问GitHub
:::
