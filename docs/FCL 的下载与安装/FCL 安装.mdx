从各个网站下载到 FCL 安装包以后，我想各位迫不及待的想安装了

但这里需要注意的是，FCL 安装过程可能会经历曲折，所以请各位小白按以下步骤去安装

1. 进入浏览器的下载页面，这里以 Chrome 浏览器为例：

![FCL文件1](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL文件1.jpg)

![FCL文件2](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL文件2.jpg)

![FCL文件3](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL文件3.jpg)

可以看到，圈出来的文件就是下载好的 FCL 安装包，请选择你的手机品牌，根据图片操作：

import BrandSteps, { BrandStep } from '@site/src/components/BrandSteps';

<BrandSteps>
  <BrandStep brand="xiaomi">
    ### 小米手机安装步骤

    点击安装包后，Chrome 会弹出提示，点击**继续**：

    ![FCL安装1](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL安装1.jpg)

    接着系统会询问是否允许 Chrome 安装应用，点击**允许**（可以顺便勾选"记住我的选择"）：

    ![FCL安装2](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL安装2.jpg)

    然后弹出 ICP 备案信息提示，直接点击**继续安装**即可：

    ![FCL安装3](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL安装3.jpg)

    然后进入安装页面，会看到"增强防护提示"，说该 App 涉诈风险较高：

    ![FCL安装4](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL安装4.jpg)

    到了这里，有些小白可能会心里一紧——"我下载的不会是假 FCL 吧？"

    先别急，这里说明一下：**FCL 启动器** 是一款开源软件，所有源代码都在 GitHub 上**公开可查**，不存在任何欺诈行为。

    > 毕竟作为一款开源工具，它的定位和社区属性决定了不会去做 [ICP 备案](https://baike.baidu.com/item/ICP%E5%A4%87%E6%A1%88/1220667)（而且备案流程本身也确实比较繁琐）。

    所以跟着图中的箭头走，点击右上角的**三个点**菜单，选择**单次安装授权**：

    ![FCL安装5](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL安装5.jpg)

    然后就会开始安装，等待进度完成：

    ![FCL安装6](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL安装6.jpg)

    等待 **FCL 启动器** 安装完毕，点击下方的**完成**即可：

    ![FCL安装7](/img/docs/FCL%20的下载与安装/FCL%20安装/FCL安装7.jpg)

    :::tip 总结
    小米手机的 FCL 安装流程：**继续** → **允许** → **继续安装** → **单次安装授权** → **完成**。
    :::
  </BrandStep>

  <BrandStep brand="vivo">
    ### vivo 手机安装步骤

    1. 点击 FCL 安装包，若提示"禁止安装"，点击**设置** → 允许来自此来源的应用
    2. 返回安装界面，点击**继续安装**
    3. 若弹出风险提示，点击**了解风险** → **仍要安装**
    4. 等待安装完成，点击**完成**

    :::caution 注意
    vivo 部分机型会弹出"该应用未经过安全检测"的警告，这是正常现象，点击**仍要安装**即可。
    :::
  </BrandStep>

  <BrandStep brand="oppo">
    ### OPPO 手机安装步骤

    1. 点击 FCL 安装包
    2. 若提示"禁止安装未知来源应用"，点击**设置** → 开启允许开关
    3. 返回并点击**继续安装**
    4. 若提示"高危应用"，点击**忽略风险安装**
    5. 安装完成后点击**完成**
  </BrandStep>

  <BrandStep brand="huawei">
    ### 华为手机安装步骤

    1. 点击安装包，若提示"发现恶意应用"，点击**了解更多** → **仍然安装**
    2. 若开启了纯净模式，需先关闭（设置 → 系统和更新 → 纯净模式 → 退出）
    3. 按提示完成安装
  </BrandStep>

  <BrandStep brand="honor">
    ### 荣耀手机安装步骤

    与华为类似，荣耀手机的 MagicOS 系统也可能拦截第三方应用：

    1. 点击安装包，选择**继续安装**
    2. 若提示风险，点击**了解风险** → **仍要安装**
    3. 安装完成后即可使用
  </BrandStep>

  <BrandStep brand="samsung">
    ### 三星手机安装步骤

    三星 One UI 对第三方应用安装限制较少：

    1. 点击安装包，若提示"未知来源"，点击**设置** → 允许此来源
    2. 返回安装界面，点击**安装**
    3. 等待安装完成
  </BrandStep>

  <BrandStep brand="oneplus">
    ### 一加手机安装步骤

    1. 点击安装包
    2. 若出现"禁止安装"提示，点击**设置** → 开启允许开关
    3. 返回点击**继续安装** → **安装**
    4. 完成
  </BrandStep>

  <BrandStep brand="meizu">
    ### 魅族手机安装步骤

    1. 点击 FCL 安装包
    2. Flyme 系统可能会提示"应用来源未知"，点击**继续安装**
    3. 若出现安全扫描，等待扫描结束后点击**安装**
    4. 安装完成
  </BrandStep>

  <BrandStep brand="realme">
    ### realme 手机安装步骤

    realme UI 与 OPPO ColorOS 同源，安装流程相似：

    1. 点击安装包
    2. 允许安装未知来源应用
    3. 忽略风险警告，点击**继续安装**
    4. 完成安装
  </BrandStep>
</BrandSteps>

---

安装完成后，回到桌面就能看到 **FCL 启动器** 的图标了，点击即可启动！
