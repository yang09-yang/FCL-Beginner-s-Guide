# 介绍

欢迎各位新人来看FCL教程

本教程专门为了方便各位理解而做的一个新手教程文档

## 前言

或许有人好奇，我为啥要创建一个教学文档。

首先，网上的各个教程内容不同，所介绍的侧重点不同（比如有的教程重点介绍联机方法，有的教程重点介绍安装方法），这就导致当新手小白需要浏览多个视频和文章才能勉强掌握，比较耗费时间。

其次，大多接触 FCL 的玩家基本上是从网易我的世界转来的，由于网易我的世界的操作简单、游玩方便，导致这类玩家转入 FCL 会出现不适应的情况，遇到问题时又不懂得如何解决。

该文档的初衷是为了让新手小白逐渐适应 FCL 的操作方式，提升解决问题的效率，从而更好地游玩手机 Java。

:::danger Warning
由于 FCL 上手难度较大，如果你第一次接触 FCL，请认真并耐心地把该文档的**"手机小白必看"**、**"FCL 的下载与安装"**、**"FCL 的使用"**、**"Minecraft 新手指引"**看完。
:::

该文档与其它教程不同的是，有"**手机小白必看**"和"**进阶教程**"的模块，前者是为没有计算机基础的玩家写的，后者是为有更多需求的玩家写的，努力让该文档做到"**一条龙服务**"的教程。

由于该文档创立不久，可能会出现缺漏与错误，也有可能在 FCL 版本更新后未能校正其内容。如有出现，欢迎大家通过GitHub或通过 QQ 群进行反馈。
